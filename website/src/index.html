Vertigo Tecnologia - Estágiario

Projetos

Transformação Cultural na Engie (DevOps)

Participação na transformação cultural para a maturidade Cloud Native, promovendo a adoção de práticas e ferramentas modernas, melhorando a colaboração entre equipes e impulsionando a eficiência e a resiliência das operações de TI.

Automação e Gerenciamento de AWS EKS e Kubernetes com Terraform

Provisionamento automatizado de VPC, Remote State Lock e EKS na AWS, implementando o gerenciamento eficiente do Kubernetes através do Terraform. Este projeto focou na aplicação de melhores práticas de segurança e gerenciamento de usuários, além de otimizar o deploy de ferramentas com Helm e a organização por namespaces. O resultado foi uma infraestrutura mais segura, ágil e fácil de gerenciar, alinhada às necessidades do negócio.

Implementação de Website Estático com CloudFront, S3 e Terraform

Desenvolvimento de um website estático econômico e eficiente utilizando a combinação de CloudFront e S3 da AWS, gerenciado por meio do Terraform. Este projeto permitiu a rápida implantação de um website resistente e de alta disponibilidade, otimizando os custos e aumentando a eficiência operacional.

Implementação de Remote State Lock com Terraform

Desenvolvimento de um sistema de bloqueio de estado remoto utilizando Terraform para garantir a consistência e evitar conflitos durante a modificação da infraestrutura. Este projeto aumentou a segurança e a eficiência da gestão da infraestrutura, permitindo que múltiplas equipes trabalhassem simultaneamente sem risco de sobrescrever alterações.

Load Stress Tool (Python)

Desenvolvimento de uma ferramenta de teste de carga e estresse utilizando Python, permitindo a avaliação precisa da robustez e resiliência dos sistemas sob altas cargas de trabalho. O projeto aumentou a confiabilidade do software, identificando pontos fracos e melhorando a capacidade de resposta da infraestrutura em situações de alta demanda.

Gitlab Migration Tool (Python)

Desenvolvimento de uma ferramenta de migração automatizada utilizando Python para facilitar a transferência de mais de 200 repositórios, 50 grupos e 50 usuários do GitLab.com para um ambiente corporativo de GitLab. A ferramenta reduziu significativamente o tempo de migração para cerca de 1 hora, minimizando a interrupção do trabalho e maximizando a eficiência operacional.

Windows DevOps Automation Tool

Desenvolvimento de uma ferramenta automatizada em PowerShell para configurar um ambiente Windows com as principais ferramentas DevOps. Este projeto simplificou e acelerou a preparação de ambientes de desenvolvimento, permitindo que os desenvolvedores se concentrassem em suas tarefas principais, aumentando assim a produtividade e a eficiência.

Kubernetes HPA: Custom Metrics for Effective CPU & Memory Scaling

Este artigo explora o uso de métricas personalizadas no Kubernetes Horizontal Pod Autoscaler (HPA) para um dimensionamento eficaz de CPU e memória. O texto oferece uma visão detalhada de como aproveitar ao máximo o HPA, garantindo um uso otimizado de recursos e melhorando a eficiência e a resiliência do sistema.

Acelerando a Transformação Digital: CALMS, Three Ways e a Jornada Cloud Native
